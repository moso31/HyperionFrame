# HyperionFrame
HyperionFrame 是一个基于离线渲染的框架。目前最新的版本是 0.3.1。

## 操作说明
在 Debug 版本下下，点击鼠标左键可以获取该位置的光线追踪具体信息。
使用 G 键可以渲染图片，对渲染数据的存储采用了大量的 STL 迭代器，**如果使用 Debug 渲染图片，速度会很慢很慢。推荐在 Release 下进行渲染。**

## 功能支持

### 积分器：
目前只支持 Whiited 积分器。

### 材质：
支持哑光（matte）材质，镜面（mirror）材质和玻璃（glass）材质。

### 灯光：
目前只支持点光源。

### 形状：
目前只支持 Box 和 Sphere。
